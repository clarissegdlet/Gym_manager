**Smart Gym Manager** is a web application designed for small and medium-sized fitness centers.  
It centralizes the daily management of a gym, avoiding the need to use multiple separate tools.

The application is primarily intended for **gym managers and administrators**.

The system manages the full day-to-day activity of a gym, from member registration to class participation.
